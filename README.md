# Dark Star Menu <img src="https://cdn.kastatic.org/images/badges/eclipse.png" alt="Black Hole Badge From Khan Academy" width="64">

**A Khan Academy Utility Mod Menu**

Dark Star Menu is a userscript menu for Khan Academy that adds features that don't make your life hell on Khan Academy.

It is designed to work with **Tampermonkey** and **Violentmonkey**.

## Features

<details>
<summary><strong>Bypass Interaction</strong></summary>

Bypasses the drawing board so you can input answers while also drawing.

This is useful when Khan Academy's drawing board blocks interaction with answer boxes.

</details>

<details>
<summary><strong>Force Calculator</strong></summary>

Opens a calculator directly from the Dark Star Menu.

This is useful when Khan Academy does not automatically show a calculator, or when you want quick access to one.

</details>

<details>
<summary><strong>Open Video</strong></summary>

Opens the related Khan Academy lesson video when one is available.

This helps you quickly jump to the video connected to the current exercise or lesson.

</details>
